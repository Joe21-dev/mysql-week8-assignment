# mysql-week8-assignment


```
  #Basic student portal
  This project allows users to enter there inforamtion which is sotred ih the database

  To run this project locally
1. Install dependencies:
  pip install fastapi uvicorn mysql-connector-python
2. Create the database:
  CREATE DATABASE student_portal;
3. Run the API:
  uvicorn main:app --reload
4. Open UI:
  Double-click index.html in your browser OR host it with a simple server (e.g., python -m http.server).
5.Test Endpoints:
  Visit: http://127.0.0.1:8000/docs for interactive Swagger UI.

```
